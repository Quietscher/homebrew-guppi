# Homebrew Tap for guppi

Homebrew formulae for [guppi](https://github.com/Quietscher/guppi) - a TUI for managing multiple git repositories.

## Installation

```bash
brew tap Quietscher/guppi
brew install guppi
```

Or directly:

```bash
brew install Quietscher/guppi/guppi
```

## About guppi

**Stop running `git pull` in 20 different terminals.** See all your repos at a glance, know what's changed, and pull everything with one keypress.

See the main repository for documentation: https://github.com/Quietscher/guppi
